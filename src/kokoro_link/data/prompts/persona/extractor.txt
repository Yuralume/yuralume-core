==== 嚴格的反幻覺規則（違反一律丟棄）====
1. quote 必須從本輪「${operator_label}說」中**逐字擷取**，不可使用近期脈絡的舊句子。
2. quote 必須來自 ${operator_label}，角色的話絕對不能當證據。
3. subject 請判斷 quote 說的是誰：operator_self / other_person / character / unclear。**每一筆候選都必須給 subject。**
4. Layer 3 (情感心理) 必須是對方**第一人稱**陳述（「我...」、「我覺得...」），角色推測對方情緒不算。
5. Layer 5 (信任依賴) 必須是對方**明確說出**具體事件，請額外標 explicit=true；隱晦暗示一律不收。
6. Layer 3 / Layer 5、以及 Layer 1 的 name / nickname，其 subject 必須是 operator_self，否則不要輸出。
   - **name / nickname 描述的永遠是「${operator_label} 這個人本身」叫什麼**，只有當對方在說自己叫什麼、或希望你怎麼稱呼「他自己」時才成立（例：「我叫…」「叫我…」「我的名字是…」）。
   - 對方在對話中**提到的別人**——朋友、家人、其他角色（即使你認識那個角色、即使名字就出現在本輪訊息裡）——的名字，subject 一律是 other_person / character，**絕對不可**當成 ${operator_label} 的 name / nickname。例：${operator_label} 說「我跟小美很熟」「角色B 也會來」「我同事阿凱…」→ 不要輸出任何 name / nickname 候選，因為小美 / 角色B / 阿凱 都不是 ${operator_label} 本人。
7. 其他欄位若 quote 明顯在講第三人（subject=other_person / character），也不要把它當成 ${operator_label} 自己的事實。
8. 不確定 → 不要寫。空陣列也是正確答案。
9. 不要重複輸出 [已知的畫像] 中已存在的欄位，除非你帶來新的事實或更新。
10. value 是玩家可見畫像會顯示的內容，必須使用上方指定的玩家可見輸出語言（或保留對方原句所用的語言），不要擅自把英文/日文陳述翻成中文；quote 一律逐字保留原文，不受此規則影響。
