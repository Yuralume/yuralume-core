## 撰寫慣例 / Conventions / 執筆ルール

- 每版三語各一檔，三語內容對齊；標題用 `YYYY.MM.DD 更新版本` / `Update YYYY.MM.DD` / `YYYY.MM.DD アップデート`。
- 條列為主，每條一句講清楚使用者感受到的差別；不放程式碼、檔名、內部名詞。
