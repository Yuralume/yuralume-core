# Update 2026.07.13

1. Character-to-character conversations got a big quality upgrade: fixed an issue where characters kept rehashing the same topic every time they met. They now bring their own lives into the conversation — today's schedule, their goals, the story they're in, the weather, recent moments with you (shared only as deep as the closeness warrants) — and they remember what they've already covered so they don't repeat it. Overall quality is now held to the same standard as conversations with you.
2. Secondhand news no longer hardens into "fact": gossip passed between characters is marked as hearsay, stays out of the public feed, and won't be treated as something the character experienced firsthand.
3. Sending photos to a character is more reliable: previously, if the currently selected model couldn't understand images, sending a photo simply errored out. Now the system automatically hands the image to a model that can see, and you can pin which model handles vision in the admin console. If your setup genuinely has no vision-capable model, the character will naturally tell you they can't make out the picture — instead of the conversation breaking.
4. Characters can now be frozen: characters you've stopped interacting with can be frozen by hand in the admin console, or automatically after an idle period you choose. A frozen character pauses all background activity (proactive messages, social life, feed posts) and stops quietly running up costs; sending them a message wakes them instantly. Idleness is judged only by your last real interaction — a character's own background activity doesn't count.
5. Fuller cost tools in the admin console: a new per-character usage cost report shows at a glance who costs the most, plus a cost-modeling tool that projects future spending from your actual usage.
6. Reasoning effort can be set per feature: the same model can now run with different reasoning strength for different jobs — light and quick for chat, deep and careful for story planning — instead of one setting for everything.
7. A new option for web lookups: characters can now use OpenAI's built-in web search.
8. A few small fixes: when a tool lookup fails, raw technical output no longer shows up in the conversation; the creation wizard's checks on short-piece themes and reasoning-strength settings are more reliable.
