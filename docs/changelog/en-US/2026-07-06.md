# Update 2026.07.06

1. The AI relationship suggestions during character creation are steadier: pressing "suggest again" several times in a row no longer errors out or wipes the suggestions you were already shown, so you can keep trying.
2. Bring in your existing character cards: you can now import common character-card formats, including the PNG cards that keep a character's data embedded inside the image — no retyping from scratch to bring an old character to life here.
3. Characters can look things up on the web: when they need to, they can reach information beyond their training data. The default is a search backend you can self-host and that's easier on your privacy, so your queries don't quietly become someone else's data.
4. More control over how a model works: you can hide the "thinking out loud" stream some models produce (off by default, nothing changes unless you turn it on), tune how hard a model reasons, and — for advanced users — pass extra parameters yourself.
5. More models and image options: more model providers are supported, and you can connect directly to an image-generation tool running on your own machine.
6. More consistent language: fixed spots where the creator studio's story arcs, provider options, and a character's "current intent" still showed Chinese on English and Japanese interfaces; the Japanese interface also got a large-scale cleanup.
7. Self-host settings are easier to reach: tool settings like weather, calendar, search, and content filtering can now be changed right in the admin console, instead of editing a config file and restarting the whole stack.
8. Names no longer get flipped: fixed an issue where calling a character directly in chat (say "big brother," "honey," or the character's own name) was occasionally misread as "please address me this way from now on," making the character call you by that term and overwrite the name you had set. That direction inversion is now blocked, and a name you set yourself in the settings is always honored exactly as you entered it.
9. Old mixed-up nicknames now heal themselves: on top of blocking new flips, the character now checks its notes while it "sleeps" — using a stronger model to compare the name you confirmed in settings against what actually happened in your conversations, and quietly clearing out any leftover, reversed nickname so it goes back to addressing you correctly. This only adjusts the character's internal understanding; it never rewrites or deletes your past chats or memories, and the name you confirmed always wins.
