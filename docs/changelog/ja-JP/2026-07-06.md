# 2026.07.06 アップデート

1. キャラクター作成時のAI関係性サジェストが安定しました：「もう一度提案」を続けて押しても、急にエラーになったり、すでに表示されたサジェストが消えたりしなくなり、安心して何度でも試せます。
2. 手持ちのキャラクターカードを取り込めます：一般的なキャラクターカード形式（キャラ情報を画像内に埋め込んだPNGカードを含む）のインポートに対応。最初から打ち直さなくても、これまでのキャラをここで動かせます。
3. キャラクターがウェブで調べられます：必要なときに学習データの外にある新しい情報にアクセスできます。既定では自分で構築できてプライバシーに配慮した検索方式を使うので、あなたの検索が誰かのデータに変わってしまうことはありません。
4. モデルの動きをより細かく制御：一部モデルの「考えながら話す」思考の流れを隠せます（既定はオフで、既存の体験は変わりません）。推論の強さを調整でき、上級者は追加パラメータも自分で指定できます。
5. モデルと画像生成の選択肢が拡大：対応するモデルプロバイダーが増え、手元のマシンで動く画像生成ツールに直接つなげられます。
6. 言語表示の一貫性を改善：クリエイター専用エリアのストーリーアーク、プロバイダー選択肢、キャラクターの「現在の意図」など、英語・日本語UIで中国語が出ていた箇所を修正。日本語UI自体も大規模に見直しました。
7. セルフホストの設定が見つけやすく：天気・カレンダー・検索・コンテンツフィルターなどのツール設定を、設定ファイルを編集してスタック全体を再起動せずとも、管理コンソールから直接変更できます。
8. 呼び方が逆にならないように：会話の中でキャラクターを直接呼んだとき（「お兄ちゃん」「あなた」やキャラクターの名前など）に、まれに「今後わたしをそう呼んで」と誤解し、キャラクターが逆にあなたをその呼び方で呼び、あなたが設定した呼び方を上書きしてしまう問題を修正しました。この向きの取り違えを防ぐようにし、設定であなた自身が入力した呼び方は常にそのとおりに扱われます。
9. 昔の取り違えた呼び方が自動で直ります：新しい向きの取り違えを防ぐだけでなく、キャラクターは夜間の記憶整理のときに、より高性能なモデルであなたが設定で確認した呼び方と実際の会話を照らし合わせ、過去に残ってしまった逆向きの呼び方をそっと取り除いて、正しい呼び方に戻します。この処理はキャラクターの内部的な認識を調整するだけで、あなたの過去の会話や思い出の内容を書き換えたり削除したりすることはなく、あなたが確認した呼び方が常に最終的な基準になります。
