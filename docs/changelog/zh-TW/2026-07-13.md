# 2026.07.13 更新版本

1. 角色之間的對話品質大升級：修正了角色互相碰面時，一直重聊同一個話題的問題。現在他們會把各自的生活帶進對話——今天的行程、目標、正在進行的故事、天氣、最近和你相處的點滴（分享多深依親密度而定），聊過的話題也會記得別再重聊。整體品質比照角色和你對話的同一套標準。
2. 聽來的消息不再變成「事實」：角色之間轉述的八卦會標記為傳聞，不會流入公開動態，角色也不會把它當成親身經歷。
3. 傳圖片給角色更穩定：以前如果目前選用的模型看不懂圖，傳圖會直接出錯。現在系統會自動改派看得懂圖的模型，後台也可以直接指定由哪個模型負責看圖；如果你的環境真的沒有任何能看圖的模型，角色會自然地告訴你他看不到圖片，而不是整段對話壞掉。
4. 角色可以「凍結」了：不再互動的角色可以在後台手動凍結，也可以設定閒置多久就自動凍結。凍結中的角色會暫停所有背景活動（主動訊息、社交、動態發文），不再默默消耗費用；只要再傳訊息給他，就會立刻醒來。閒置的判斷只看你上次真正的互動，角色自己的背景活動不算數。
5. 後台成本工具更完整：新增每個角色的用量成本報表，一眼看出誰花最多；還有成本試算工具，可以用實際用量推估之後的花費。
6. 推理強度可以按功能分開設定：同一個模型，可以在不同功能上配不同的推理強度——聊天輕快、劇情規劃深思，不再是一套設定套全部。
7. 角色上網查資料多了新選項：新增支援 OpenAI 內建的網頁搜尋。
8. 幾個小修正：工具查詢失敗時，不再把技術性的原始內容顯示在對話裡；創作精靈的短篇主題與推理強度設定檢查更可靠。
