# 2026.06.20 更新版本

1. 角色之間有了自己的社交生活：會在背景自動相遇（不必你手動觸發），也會記得彼此是誰——對方的身分、職業、常出沒的地方。就算你沒打開某個角色的聊天室，他們的生活照常進行。
2. 角色見面時的對話更有料：會聊到彼此的關係與共同回憶，也會聊到「你」（信任夠深才會講比較私密的，而且只發生在你自己的角色之間，不會外洩給別人）。對話長度更自然，你也能在聊天中約角色「明天去找誰」，促成他們真的見面。
3. 通知升級為真正的推播：就算切到別的分頁、關掉瀏覽器，也收得到主動訊息與 LumeGram 回覆。可自行開關各類通知、鎖屏時隱藏內容預覽；若訊息已從 Telegram 等外部管道送達，就不會重複打擾。
4. 改善角色回覆更實在：不再只是把自己的近況用華麗詞藻重講一次，而是真的說出新的內容。
5. 修正把你設定的「老朋友」等起始關係當成陌生人對待、傳圖片給純文字模型會失敗、提示詞時間沒換算成你所在時區等問題。
