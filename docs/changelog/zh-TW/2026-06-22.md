# 2026.06.22 更新版本

1. 角色更記得自己做過的事：你問「今天做了什麼」，他們能講出今天稍早實際做過的事，而不是一臉茫然；行事曆上「已留下回憶」的標記，現在只會在真的有對應回憶時才顯示。
2. 不再憑空冒出「你們一起做過」的回憶：角色想找你一起做某件事時，會先在聊天裡開口問、等你答應，才會把它當成共同回憶；你沒答應的，不會被當成已經發生——避免角色一口咬定你們做過、害你以為自己記錯的情況。
3. 一起走過的主線劇情會留下痕跡：你和角色共同經歷的關鍵情節，之後會被記得並在對話中延續，重要的高潮會成為你們的關係里程碑，而不是船過水無痕。
4. 角色說話更看場合：日常閒聊時不再動不動就用心理諮商式的溫柔腔（油腔滑調），同一套溫柔措辭也不會被高頻重複；但當你真的在訴說脆弱、需要被接住時，那份溫柔仍然在。
5. 推播通知更穩定：修正了部分情況下的送達問題，並讓裝置從休眠喚醒後仍能補收，減少漏接。
