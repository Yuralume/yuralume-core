# 2026.07.06 更新版本

1. 建立角色時的 AI 關係建議更穩定：連續按「再建議一次」不會再突然出錯、把已經顯示的建議清空，可以放心多試幾次。
2. 可以帶入你既有的角色卡：支援匯入常見的角色卡格式，包含把角色資料藏在圖片裡的 PNG 卡，不必從頭重打就能讓舊角色在這裡活起來。
3. 角色可以上網查資料：需要時能查到訓練資料以外的新資訊；預設走可自行架設、較重視隱私的搜尋方式，你的查詢不會白白變成別人的數據。
4. 更能掌控模型怎麼運作：可以把部分模型「邊想邊講」的思考過程隱藏起來（預設關閉，不影響現有體驗）、調整推理強度，進階使用者還能自訂額外參數。
5. 更多模型與生圖選擇：串接了更多模型供應商，也能直接連上本機的生圖工具。
6. 語言呈現更一致：修正了創作專區的劇情弧、供應商選項、角色「當前意圖」等在英日語介面下仍冒出中文的地方；日語介面也做了大規模校潤。
7. 自架設定更好找：天氣、行事曆、搜尋、內容過濾等工具設定，現在可以直接在後台調整，不用再改設定檔、重啟整套服務。
8. 稱呼不會再被叫反：修正了一個問題——當你在對話中直接喊角色（例如喊「哥哥」「老公」或角色的名字），系統偶爾會誤把那當成「你要角色以後這樣叫你」，害角色反過來用那個稱呼叫你、蓋掉你原本設定的稱呼。現在會擋掉這種方向反轉，你在設定裡親手填的稱呼一律照你的意思，不受影響。
9. 舊的叫錯稱呼會自動修好：除了擋下新的方向反轉，角色現在會在夜間整理記憶時，用較強的模型比對你在設定裡確認過的稱呼與實際對話，把過去殘留、被叫反的暱稱悄悄清掉，讓角色重新用對的方式稱呼你。這個過程只調整內部認知、不會改寫或刪除你的歷史對話與回憶內容，你確認過的稱呼永遠是最終依據。
